Get Metadata for Multiple Videos

Request Headers

Api-Key string Provide your API key. Required.
Accept-Language string Specifies the language of result values.
Authorization string Provide access token in the format of 'Bearer: {token}'.

Querystring Parameters

ids array Specifies one or more video ids to return. Use comma delimiter when requesting multiple ids. Required.
fields array Specifies fields to return. Defaults to 'summary_set'. Possible values: id allowed_use artist asset_family caption clip_length collection_code collection_id collection_name color_type copyright comp date_created detail_set display_set download_sizes downloads era editorial_segments keywords license_model mastered_to originally_shot_on preview product_types shot_speed source summary_set thumb title

Response Body Fields

videos_detail

Videos array [video_detail] Contains an entry for each video found
images_not_found array [string] A list of video identifiers not found

video_detail

id string Identifies the video
allowed_use Specifies how a customer is allowed to use this video
artist string Identifies the artist
asset_family string Indicates the asset family classification. Possible values are: creative, editorial
caption string Describes the video
clip_length string Specifies the length of the film clip (e.g., "00:00:12:08")
collection_id integer Identifies the collection which this video belongs to
collection_code string Identifies the collection code which this video belongs to
collection_name string Identifies the collection name which this video belongs to
color_type string Specifies if it is color or black and white. Possible values are color, black_and_white, and null
copyright string Specifies the copyright
date_created string Specifies the date the video was created in ISO 8601 format (e.g., 1999-12-31T00:00:00-00:00)
display_sizes array [display_size] List of display size information
download_sizes array [download_size] List of available download size information
editorial_segments array [string] List of editorial segments. Possible values are archival, entertainment, news, publicity, royalty, sport
era string The era of the video. Possible values are: archival, contemporary, and null
keywords array [keywords] Contains a keyword entry for each keyword related to the image.
license_model string Identifies a licensing model for the video. Possible values are: royaltyfree, rightsready
mastered_to string Specifies video format of the master
originally_shot_on string Specifies where this video was originally shot on
product_types array [string] License agreements for the video. Possible values are: premiumaccess, easyaccess, editorialsubscription, imagepack, royaltyfreesubscription
shot_speed string The speed the video was shot. Possible values are: real_time, slow_motion, time_lapse, and null
source string
title string Specifies the title of this video

allowed_use

how_can_i_use_it string Specifies how the asset can be used.
release_info string Specifies the release status.
usage_restrictions array [string] Specifies asset usage restriction.

display_size

is_watermarked boolean Indicates if the asset is watermarked
name string Specifies the name of the display size
uri string Specifiest the asset URI

download_size

bit_depth string Specifies the bit depth of the video
broadcast_video_standard string Indicates broadcast video standard of the video (e.g., NTSC, PAL)
compression string Specifies video compression
content_type string Indicates content type of the video (e.g., "video/mp4")
description string Description of video
format string Specifies the video format (e.g., SD, HD, 4K, Multimedia, or Web)
frame_rate number Specifies the video frame rate (e.g., 24, 25, 29.97)
frame_size string Specifies the video frame size (e.g., "640x360")
height number Specifies the height in pixels
interlaced boolean Indicates whether the video is interlaced
name string Specifies the name of the video size
width integer Specifies the width
downloads array [download] List of video download information

download

uri string Specifies the URI for initiating download

keywords

keyword_id string
text string
type string
relevance integer
