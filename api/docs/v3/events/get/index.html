Get Metadata for Multiple Events

Request Headers

Api-Key string Provide your API key. Required.
Accept-Language string Specifies the language of result values.
Authorization string Provide access token in the format of 'Bearer: {token}'.

Querystring Parameters

fields array A comma separated list of fields to return in the response. Possible values: id child_event_count editorial_segments hero_image image_count location name start_date type
ids array A comma separated list of event ids.

Response Body Fields

events_result

events array [event]
events_not_found array [integer]

event

id integer Identifies the event.
child_event_count integer Indicates the count of child events.
editorial_segments array [string] List of editorial segments. Possible values are archival, entertainment, news, publicity, royalty, sport
hero_image Identifies the image which best represents the event.
image_count integer Indicates the number of images related to the event.
location Indicates the geographic location where the event took place.
name string Indicates the name of the event.
start_date string Indicates the date the event started.

hero_image

id string Identifies the image.
display_sizes array [hero_display_size] Specifies the list of available display sizes.

hero_display_size

name string Identifies the name of the display size.
is_watermarked boolean Indicates whether the image is watermarked.
uri string GET this URI to retrieve the display size image.

location

city string Identifies the city where the event took place.
country string Identifies the country where the event took place.
state_province string Identifies the state or province where the event took place.
venue string Identifies the venue where the event took place.

Response Status Codes

400 InvalidParameterValue
400 MalformedRequest
404 EventNotFound
