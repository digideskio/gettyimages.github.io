Getting Started

  1. Create your free account

    Register for a free account on our API site to request an API key

  2. Get Your API Key

    Which key should I choose?

    Test Key: Use Getty Images API to test all operations of the API (image search, metadata retrieval, and account management) for a selected pool of over 11 million images. This key does not offer access to downloadable images; if you're interested in download options or access to more content, please contact us via the form on our site.

    Embed Key: Use Getty Images API to search over 55 million images available for non-commercial embedding in your website, application, or social media.

  3. Try out the API

    Use our interactive tool to execute all of the operations of the API without writing a line of code.

  4. Additional details

    We also have SDKs, detailed technical documentation, and workflows with sample code to help you get started developing your application.
