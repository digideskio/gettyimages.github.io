age_of_people array Allows for filtering based on age of individuals in an image. Possible values: newborn baby child teenager young_adult adult adults_only mature_adult senior_adult 0-1_months 2-5_months 6-11_months 12-17_months 18-23_months 2-3_years 4-5_years 6-7_years 8-9_years 10-11_years 12-13_years 14-15_years 16-17_years 18-19_years 20-24_years 20-29_years 25-29_years 30-34_years 30-39_years 35-39_years 40-44_years 40-49_years 45-49_years 50-54_years 50-59_years 55-59_years 60-64_years 60-69_years 65-69_years 70-79_years 80-89_years 90_plus_years 100_over
artists string Free-text search, comma-separated artist search query.
collection_codes array Allows filtering by collection code.
collections_filter_type string Allows searching based on included or excluded collections. Possible values: include exclude
compositions array Allows for filtering based on image composition. Possible values: abstract candid close_up copy_space cut_out full_frame full_length headshot looking_at_camera macro portrait sparse still_life three_quarter_length waist_up
date_from string If specified, include only images created on or after this date. Dates should be submitted in ISO 8601 format (i.e., YYYY-MM-DD).
date_to string If specified, include only images created on or before this date. Dates should be submitted in ISO 8601 format (i.e., YYYY-MM-DD).
embed_content_only boolean Restrict search results to embeddable images. The default is false.
ethnicity array Filters search results based on the ethnicity of individuals in an image. Possible values: black caucasian east_asian hispanic_latino japanese middle_eastern mixed_race_person multiethnic_group native_american_first_nations pacific_islander south_asian southeast_asian
event_ids array Allows for searching based on specific events
exclude_nudity boolean Exclude images tagged as containing nudity. The default is false.
file_types array Identifies a file type for the image. Possible values: eps gif jpg png
graphical_styles array Identifies a graphical style for the image. Possible values: fine_art illustration photography
keyword_ids array Comma separated-list of keyword IDs. If specified, returns images containing the requested keyword(s). If keyword IDs and phrase are both specified, returns only those images matching the query phrase which also contain the requested keyword(s).
license_models array Identifies a licensing model for the image. Possible values: rightsmanaged royaltyfree
number_of_people array Allows for filtering based on the number of people in the image Possible values: none one two group
orientations array Choose images based on their aspect ratios. Possible values: Horizontal Vertical Square PanoramicHorizontal PanoramicVertical
phrase string Free-text search query.
prestige_content_only boolean Restrict search results to prestige images. The default is false.
product_types array Allows filtering by a list of product types. Possible values: easyaccess editorialsubscription imagepack premiumaccess royaltyfreesubscription
specific_people array Allows for filtering based on specific people keywords.
